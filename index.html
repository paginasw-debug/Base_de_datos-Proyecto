Sistema de Base de Datos para la Biblioteca

Unidad Educativa Fiscal Dolores Sucre

En la actualidad, las bibliotecas deben adaptarse a los avances tecnológicos para ofrecer un servicio más ágil y eficiente. Este proyecto propone el diseño de una base de datos que permita ordenar y controlar los ejemplares disponibles, facilitando la búsqueda y el préstamo de libros a estudiantes y docentes.

Con esta herramienta se busca modernizar la administración bibliográfica, garantizando un acceso rápido y confiable a los recursos educativos.

Sobre el Proyecto

Memoria Descriptiva

Este proyecto tiene como objetivo el diseño y desarrollo de una base de datos para la biblioteca de la Unidad Educativa Fiscal Dolores Sucre. Su propósito es optimizar la organización de la información de los libros, usuarios y préstamos, permitiendo un acceso rápido, ordenado y eficiente para estudiantes y docentes.

El sistema fue desarrollado en Microsoft Visual FoxPro 9.0, un entorno orientado a objetos que permite crear aplicaciones robustas, seguras y con interfaz visual amigable. Gracias a esta herramienta, los usuarios podrán realizar búsquedas, registrar préstamos y gestionar la disponibilidad de ejemplares de manera automatizada.

Características Técnicas

  • Desarrollo en Visual FoxPro 9.0 con módulos de usuarios, libros y préstamos.
  • Gestión relacional entre tablas (uno a muchos) para mantener la coherencia de los datos.
  • Interfaz visual con formularios, botones de comando, y cuadros de búsqueda.
  • Integración con otros sistemas mediante ODBC (Access, MySQL, SQL Server).

Seguridad y Protección de Datos

  • Acceso mediante usuario y contraseña.
  • Copias de seguridad automáticas y cifrado de información sensible.
  • Permisos jerárquicos y bloqueo de edición simultánea.
  • Protección antivirus y firewall institucional.

Ventajas del Sistema

  • Agiliza los procesos de registro, búsqueda y préstamo de libros.
  • Evita pérdidas de información y deterioro físico de documentos.
  • Permite ampliación futura y mejora continua del sistema.
  • Fomenta el uso tecnológico dentro del entorno educativo.

Funciones del Sistema

La base de datos de la biblioteca integra diversas funciones que permiten gestionar eficientemente los libros, usuarios y préstamos, garantizando control, seguridad y accesibilidad a la información.

Gestión de Libros

  • Registro de libros con información completa: título, autor, editorial y año de publicación.
  • Mantenimiento de inventario: control de entradas y salidas de ejemplares.
  • Búsqueda rápida por título, autor, tema o código interno.

Gestión de Usuarios

  • Registro de estudiantes, docentes y personal de la biblioteca.
  • Control de acceso mediante usuario y contraseña.
  • Gestión de permisos y niveles de acceso según rol.

Préstamos y Devoluciones

  • Registro automático de préstamos y devoluciones en tiempo real.
  • Alertas de libros disponibles o en préstamo.
  • Generación de reportes de movimientos de libros.

Seguridad y Mantenimiento

  • Copia de seguridad automática y protección de datos sensibles.
  • Bloqueo de edición simultánea para evitar errores.
  • Actualización periódica de registros y ampliación de capacidad del sistema.
